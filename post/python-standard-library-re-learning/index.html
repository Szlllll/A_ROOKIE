BG：今天帮好朋友黑🦆写python的大作业，要用到re这个库，用了一下午查查资料模模糊糊地整出来了。感觉re这个库还是挺牛的，学好了或许以后不管是做项目/算法题，在字符串这一块的处理或许能有一些奇技淫巧，就打算系统地来学一学。
(RE = Regular Expression 正则表达式)

  • r'\n' contains '' and 'n'
  • \n only contains \n

Special Characters

  • .：匹配任意字符
  • ^：匹配字符串开始字符
  • $：匹配字符串结束字符
  • *：匹配任意次数前面那个字符。例如，ab*可以匹配a（b出现0次），可以匹配ab（b出现1次），也可以匹配abbb...bbb（b出现n次）
  • +：匹配1或更多次数前面那个字符。例如ab+只能匹配ab及更多次数的b
  • ？：匹配0或1次前面那个字符。例如ab?只能匹配ab或者a
  • *？、+？、？？：最后添加？表示non-greedy mode，例如[.*?]对于字符串[a] b [c]只会匹配一个[a]，而[.*]则会整个都匹配（最长）
